web7

代码审计实锤了

第一步就是利用弱类型绕过。

第二步就是数组绕过。

第三步我们看到利用了string进行强制类型转换，我们无法用数组绕过了。只能用MD5强碰撞，要求传入三个值不能相等，但是MD5相等，我们换个思路，不一定要传入三个值，可以传入文件，这里用到一个MD5碰撞的工具，用这个工具可以生成若干MD5相等的文件，传入就可以了。

https://github.com/thereal1024/python-md5-collision

再往后就是

需要传入a、b、m、n参数，$a为大写字母，$b为数字、并且长度为6，$m和$n长度小于4
$str8为$ahash加密后的值，$str9为$bhash加密后，把$m替换为$n,要求$str8== $str9。还是用的弱类型。
传入的$a要hash加密后为0e开头，如果$b为hash加密后0e开头，可以满足，长度为6不能满足，但是还有次替换，只需要把0e后不是数字的替换为数字就可以。

该死的温柔

题目的名字叫该死的温柔，那么我们可以发现该死和guess谐音，那么我们有个隐写工具OutGuess，它是一种通用的隐写工具，可以插入将隐藏信息存入数据源的冗余位。
Kail终端命令输入：

git clone https://github.com/crorvick/outguess

下载后使用

outguess -k “guess” -r flag.jpg ctf.txt

其中guess是密码在照片属性哪里，-r是解密指令，ctf.txt是输出ctf.txt文件
