python整数类型

四种进制表示形式

  • 十进制：1010,99，-217
  • 二进制：以0b或0B开头：0b010，-0B101
  • 八进制：以0o或0O开头：0o123，-0O456
  • 十六进制：以0x或者0X开头：0x9a，-0X89

python浮点数类型

浮点数之间的运算存在不确定尾数，不是bug

比如0.1+0.20.30000000000000004

解决方法：round(x,d):对x四舍五入，d是小数截取位数

使用字母e或E作为幂的符号

e 表示a*10b次方

如4.3e-3的值为0.0043

python负数类型

若x的平方-1，哪x的值是多少？
所以定义j根号下-1，以此为基础构造负数
a+bj称为负数，a是实部，b是虚部

实例：z1.2e-4+5.6e+89j

z.real 获取实部
z.imag 获取虚部

部分python函数

abs(x) 取绝对值
divmod(x,y) 商余，(x//y,x%y)同时输出商和余数
如divmod(10,3) 输出3,1
pow(x,y[,z]) 幂余，(x**y)%z,[..]表示参数z可省略
max(a,b,c…) 判断其中的最大值
min(a,b,c…) 判断其中的最小值
int(x) 将x变成整数
float(x) 将x变成浮点数

python字符串类型

-由一对单引号或双引号表示，仅表示单行字符
-由一对三引号或三双引号表示，可表示多行字符

字符串包含单引号或者双引号

‘这有个双引号”哈哈哈’
“这有个单引号’哈哈哈”

字符串操作：

使用[]获取字符串中的一个或多个字符

索引：

返回字符串中单个字符 <字符串>[M]
如 “temp：”[0] 返回‘t’

切片：

返回字符串中一段字符子串 <字符串>[M:N]

如 “temp”[1:3] 返回 ‘em’

切片高级用法

<字符串>[M:N:K],根据步长k对字符串进行切片
表示从第一个字符开始截取，间隔k个字符取一个。

如”一二三四五六七八九十”[1:8:2] 返回’二四六八’
如”一二三四五六七八九十”[::-1] 返回’十九八七六五四三二一’

字符串操作方法

str.strip( chars_rm ) 详解

（1）声明：str 为字符串，chars_rm为要删除的字符序列
（2）细分

str.strip(chars_rm) # 删除s字符串开头、结尾处，位于rm删除序列中的字符
str.lstrip(chars_rm) # 删除s字符串开头处，位于rm删除序列中的字符
str.rstrip(chars_rm) # 删除s字符串结尾处，位于rm删除序列中的字符

（3）注意：

1）当chars_rm为空时，默认删除空白符（包括’\n’, ‘\r’, ‘\t’, ‘ ‘)
2）只要边（开头或结尾）上的字符在删除序列chars_rm内，就删除掉。

（4）例如


>>> ' 78ihvyf \n'.strip()
'78ihvyf'

str.join( iter ) 详解

(1）语法： ‘sep’.join(iter)：以sep作为分隔符，将seq所有的元素合并成一个新的字符串

（2）参数说明
sep：分隔符（字符串），可以为空。
iter：要连接的元素序列、字符串、元组、字典（可迭代对象）

（3）返回值：返回一个以分隔符sep连接各个元素后生成的字符串


# 字符列表转字符串
>>> ''.join(['e', 't', 'fff13fff', 'k', '.', 'h', 'l'])
'etfff13fffk.hl'

字符串的格式化

字符串格式化使用.format()方法，用法如下：
<模板字符串>.format(<逗号分隔的参数>)
槽
例子


>>> "{}:计算机{}的CPU占用率为{}%".format("1949-10-1","CHA",10)
'1949-10-1:计算机CHA的CPU占用率为10%'

format()方法的格式控制

例子


>>> "{0:=^20}".format("0123456789")
'=====0123456789====='

>>> "{0:e}；{0:,.2f}；{0:,%}" . format(123.1415926)
'1.231416e+02；123.14；12,314.159260%'

# 使用 {{}} 输出 {}
>>> a = 123.141592
>>> b = 'ABCDEF'
>>> print( "{{}}{0:e},{1:=^10},{0:.2f}".format(a, b) )      
{}1.231416e+02,==ABCDEF==,123.14
# ':'号前面的数字为 参数序号

A.format(X)方法是字符串的方法，操作完了得到的也是字符串。但是X可以是数字或者字符串。
{}里面的标点必须是英文标点符号。
