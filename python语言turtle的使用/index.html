python语言turtle的使用

Diposting di 2018-08-18 | Di python | Views:

不要和文艺青年睡觉，否则你将会出现在他的歌里、诗里、短篇里，他们喜欢把你根本不在意的事情 描述的凄美动听，普通的文艺青年还好，要是运气不好碰到道行深的，他将告诉你——你的名字解释了他的一生。

python语言turtle库的一些小记，不得不说北京xx大学的这个python讲的真好，真心喜欢那老师的讲课风格。

空间坐标体系

绝对坐标

turtle.setup(width,height,startx,starty)
//设置绘图窗体的大小，后两个是起始点的坐标，可选

turtle.goto(x,y)
//去往某个坐标，

实例如下：


#pythondraw.py
import turtle
turtle.setup(650,350,200,200)
turtle.goto(100,100)
turtle.goto(100,-100)
turtle.goto(-100,-100)
turtle.goto(-100,100)
turtle.goto(0,0)

海龟坐标

turtle.fd()
//向海龟的正前方前进
turtle.bk()
//向海龟的反方向前进
turtle.circle(r,angle)
//以一个点为圆心，作曲线运动。
//r为距离，angle为角度

角度坐标体系

turtle.seth(angle)
//seth用来设置行进角度，但是不会进行移动
turtle.left(angle)
turtle.right(angle)
//分别设置向左和向右的角度

实例：


#pythondraw.py
import turtle
turtle.fd(300)
turtle.left(225)
turtle.fd(400)
turtle.right(225)
turtle.fd(300)

颜色模式：

turtle.colormode(mode)
-1.0：rgb的小数模式
-255：rgb的整数模式

画笔控制

画笔操作一直有效，一般成对出现

turtle.penup() 别名 turtle.pu()
//画笔抬起，
turtle.pendown() 别名 turtle.pd()
//画笔落下
turtle.pensize(width) 别名 turtle.width(width)
//画笔的宽度
turtle.pencolor(color)
//设置画笔的颜色，参数为RGB格式或颜色名称

运动控制

turtle.forward(d) 别名 turtle.fd(d)
//向前行进，d为距离，可以为负数
turtle.circle(r,angle)
//以一个点为圆心，作曲线运动。
//r为距离，angle为角度
例如：turtle.circle(100)

import的更多用法

第一种

import<库名>
<库名>.<函数名>(<函数参数>)

第二种

from<库名>import<h函数名>
from<库名>import *
<函数名>(<函数参数>)

第三种

import<库名>as<库别名>
<库别名>.<函数名>(<函数参数>)

第一种不会出现函数重名问题

python绘画玫瑰花实例


#pythondraw.py

import turtle

# 设置初始位置
turtle.penup()
turtle.left(90)
turtle.fd(200)
turtle.pendown()
turtle.right(90)

# 花蕊
turtle.fillcolor("red")
turtle.begin_fill()
turtle.circle(10,180)
turtle.circle(25,110)
turtle.left(50)
turtle.circle(60,45)
turtle.circle(20,170)
turtle.right(24)
turtle.fd(30)
turtle.left(10)
turtle.circle(30,110)
turtle.fd(20)
turtle.left(40)
turtle.circle(90,70)
turtle.circle(30,150)
turtle.right(30)
turtle.fd(15)
turtle.circle(80,90)
turtle.left(15)
turtle.fd(45)
turtle.right(165)
turtle.fd(20)
turtle.left(155)
turtle.circle(150,80)
turtle.left(50)
turtle.circle(150,90)
turtle.end_fill()

# 花瓣1
turtle.left(150)
turtle.circle(-90,70)
turtle.left(20)
turtle.circle(75,105)
turtle.setheading(60)
turtle.circle(80,98)
turtle.circle(-90,40)

# 花瓣2
turtle.left(180)
turtle.circle(90,40)
turtle.circle(-80,98)
turtle.setheading(-83)

# 叶子1
turtle.fd(30)
turtle.left(90)
turtle.fd(25)
turtle.left(45)
turtle.fillcolor("green")
turtle.begin_fill()
turtle.circle(-80,90)
turtle.right(90)
turtle.circle(-80,90)
turtle.end_fill()

turtle.right(135)
turtle.fd(60)
turtle.left(180)
turtle.fd(85)
turtle.left(90)
turtle.fd(80)

# 叶子2
turtle.right(90)
turtle.right(45)
turtle.fillcolor("green")
turtle.begin_fill()
turtle.circle(80,90)
turtle.left(90)
turtle.circle(80,90)
turtle.end_fill()

turtle.left(135)
turtle.fd(60)
turtle.left(180)
turtle.fd(60)
turtle.right(90)
turtle.circle(200,60)
